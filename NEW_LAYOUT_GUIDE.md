# New Layout Guide - Set.a.light 3D Inspired

Your photography studio has been completely redesigned with a professional layout inspired by Set.a.light 3D.

## Color Scheme

- **Background**: #F2EFE9 (warm light cream)
- **Text**: #403F3C (dark charcoal)
- **Accent**: #8B7355 (warm brown)
- Clean, professional light theme

## Layout Structure

### Top Bar
- **Aro Logo** - Brand identity on the left
- **File Menu** - File, Edit, Settings, Help, Undo buttons
- **Main Tabs** - SETUP, VIEW, EXPORT, COMMUNITY
- **Camera Settings Bar** - Professional camera controls:
  - Frame format (Full Frame, APS-C, Micro 4/3)
  - Aperture (f-stop)
  - Focal length (mm)
  - Shutter speed
  - ISO
  - Kelvin temperature
  - Color picker
- **Auth Section** - Login/signup on the right

### Left Sidebar
- Character avatar and profile
- Pose controls (IK/FK modes)
- Style, Pose, Bones tabs
- Transform controls (Position, Height, Rotation, Size)
- Hair and clothing selectors
- Preset poses library (Standing, Sitting)
- All existing lighting presets and studio controls

### Main Viewport
- Large central 3D canvas with your model
- Viewport controls (Frame All, Pan, Zoom, Settings)
- **Version tag** in bottom right corner

### Right Panel
- **TOPVIEW** tab - Overhead/top-down view of your scene
- **SETLIST** tab - List of all objects in the scene (characters, lights, props)

### Bottom Equipment Timeline
- **Timeline Controls** - Play, Stop, time display
- **Equipment Category Tabs**:
  - TIMELINE - Animation timeline (coming soon)
  - **MODELS** - All available character models from database
  - **MONOLIGHT** - Studio lights (Long Rectangular, Rectangular, Square, Round softboxes)
  - SPEEDLIGHT - Flash equipment (coming soon)
  - CONTINUOUS LIGHT - Continuous lighting (coming soon)
  - HELPER - Reflectors, flags, etc. (coming soon)
  - PROPS - Props and set pieces from database

## How to Use

### Adding Models
1. Click the **MODELS** tab in the bottom timeline
2. Click any model thumbnail to load it into the scene
3. Models are automatically loaded from your Supabase database

### Adding Lights
1. Click the **MONOLIGHT** tab in the bottom timeline
2. Click any light thumbnail to add it to the scene
3. Lights appear in the overhead view and can be positioned
4. Use the left sidebar lighting controls to adjust power, size, rotation, color

### Switching Views
1. Click the **VIEW** tab at the top to access view options
2. Switch between different viewport layouts
3. Toggle TOPVIEW and SETLIST in the right panel

### Character Customization
- Use the left sidebar character controls
- Switch between Style, Pose, and Bones tabs
- Adjust hair, clothing, and poses
- Use transform controls to position and scale

### Exporting
1. Click the **EXPORT** tab at the top
2. Access High-Quality Render, Lighting Blueprint, Setup Diagram, Equipment Specs

## Key Features

- All model selection now happens in the bottom timeline (no more sidebar clutter)
- All lighting equipment organized by type in bottom tabs
- Professional camera settings always visible in top bar
- Clean separation of character controls (left), viewport (center), scene overview (right)
